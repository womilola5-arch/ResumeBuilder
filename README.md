## What It Does

- Scans your photo library
- Identifies old and large files
- Shows storage breakdown by categories
- Lets you delete unwanted files
- Helps free up phone storage

## Tech Stack

- Flutter 3.0+
- Provider (state management)
- SQLite (local database)
- Photo Manager (file access)

## Setup

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Clone this repo
3. Run: `flutter pub get`
4. Run: `flutter run`

## How to Use

1. Open the app
2. Grant photo permissions
3. Tap "Scan Files" button
4. Review old and large files
5. Select files to delete
6. Free up storage space!
